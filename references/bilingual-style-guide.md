# Bilingual Style Guide

- 中英文保持事实、所有权、数字、日期和技术名词一致，不逐字硬译。
- 中文先写动作和产物，减少主语与“负责/参与”。
- 英文使用过去式描述完成工作、现在时描述持续系统；以强但准确的动词开头。
- 技术名词优先官方拼写；首次出现可写全称与缩写。
- 英文 bullet 避免 responsible for、helped、familiar with；若真实所有权有限，使用 contributed to、implemented part of 并说明范围。
- 占位符也双语对应，例如 `[规模待补]` / `[scale to confirm]`。

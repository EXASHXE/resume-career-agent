# Narrative Tools

## 框架

- STAR：Situation → Task → Action → Result。
- CAR：Challenge → Action → Result。
- XYZ：Accomplished X, measured by Y, by doing Z；Y 未知时保留占位符。
- 动作 + 产物 + 结果：`实现 X 模块，通过 Y，支持/减少 Z`。
- 问题 → 方案 → 权衡 → 结果：适合复杂系统设计。
- 系统定位 → 技术挑战 → 核心实现 → 影响：适合项目概述和讲稿。

## Bullet 模板

中文：`针对[问题/约束]，设计/实现[产物]，通过[关键机制/权衡]实现[结果待补]。`

英文：`Designed/Implemented [artifact] to address [problem/constraint], using [mechanism/trade-off] to achieve [result to confirm].`

每条只承载一个主结论；动词必须匹配真实所有权。没有结果证据时描述能力或产物，不推断收益。

# JD Keyword Map

用于扩展同义词，不用于硬塞关键词。

| Role | Typical domains and signals |
|---|---|
| AI Infra Engineer | distributed training, profiling, accelerator, memory, throughput, PyTorch |
| LLM Infrastructure Engineer | LLM training/serving, Megatron, parallelism, inference, GPU/NPU |
| Distributed Training Engineer | TP, PP, DP, collective communication, NCCL/HCCL, checkpoint |
| ML Systems Engineer | model runtime, compiler, performance, experiment platform, reliability |
| Backend Engineer | API, database, cache, queue, concurrency, observability |
| Platform Engineer | developer platform, control plane, automation, multi-tenancy, governance |
| Agent Engineer | agent, RAG, tool use, planning, OpenAPI, evaluation, guardrails |
| CloudOps Engineer | cloud operations, IAM, resource pool, automation, incident, API |
| DevOps Engineer | CI/CD, IaC, container, Kubernetes, release, observability |
| SRE | SLO/SLI, incident response, capacity, toil, reliability, on-call |
| Tooling / Developer Productivity Engineer | CLI, SDK, build, test, workflow, developer experience |

将 `required/must/必须` 附近条目归入 must-have；将 `preferred/plus/加分` 归入 nice-to-have。年限、lead/architect/ownership/on-call 属于 seniority signals。
